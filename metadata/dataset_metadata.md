# Housing Dataset Metadata

## Dataset Title

Predicting Housing Prices Using Real Estate Data in European Cities

---

## Dataset Description

This dataset contains structured information related to housing properties from selected European cities. The dataset is intended for machine learning experiments focused on predicting housing prices using property characteristics and geographical information.

---

## Dataset Attributes

| Attribute | Description |
|---|---|
| city_name | Name of the city |
| country_name | Country where the property is located |
| property_type | Type of residential property |
| construction_year | Year in which the property was constructed |
| total_rooms | Number of rooms |
| property_size_m2 | Property size in square metres |
| listing_price_eur | Property listing price in Euro |
| listing_date | Date of property listing |

---

## Data Source

The dataset is compiled from publicly available real-estate listing information and educational sample datasets for academic purposes.
